# Imperial_College_Capstone
The Capstone project for Imperial College's AI and ML course

Using machine learning to analyse Pokémon card market data and forecast future card values.

# Project goal

The goal of this project is to build a machine learning model that predicts the future market value of Pokémon trading cards based on historical price data and card characteristics.

By combining time-series price trends with card features such as rarity, release year, grading score and popularity, the model aims to estimate future price movements and help identify cards with strong long-term investment potential.


# Project goal

The goal of this project is to build a machine learning model that predicts the future market value of Pokémon trading cards based on historical price data and card characteristics.

By combining time-series price trends with card features such as rarity, release year, grading score and popularity, the model aims to estimate future price movements and help identify cards with strong long-term investment potential.


# Motivation

The Pokémon card market has grown significantly in recent years and behaves similarly to collectible asset markets such as sports cards or rare coins.

This project explores whether machine learning techniques can detect patterns in historical sales data and provide useful insights into future card value trends.
