# Imperial_College_Capstone
The Capstone project for Imperial College's AI and ML course

Using machine learning to analyse Pokémon card market data and forecast future card values.

# Data

The first step of this project is to gather reliable data that is relevant for ML to forecast the value of Pokemon cards.

There are multiple opportunities to gather such data. Some publicly available databases include:

- https://scrydex.com/
- https://tcgdex.dev/?utm_source=chatgpt.com
- https://www.pricecharting.com/api-documentation
- https://www.pokemon-api.com/?utm_source=chatgpt.com
- https://tcgcsv.com/
- E-bay scrapping
